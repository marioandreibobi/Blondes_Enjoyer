# Clear Session Context

This workflow clears the current session context and resets the workspace state.

## Steps:

1. Clear chat history and conversation context
2. Reset any temporary workspace variables
3. Clear any active todo lists
4. Reset memory context for the current session

## Usage:

Run this workflow with `/clear` to start fresh while keeping your MCP servers and workspace files intact.

## Notes:

- This does not delete any files from your workspace
- MCP server configurations remain active
- Only the conversational context is cleared
